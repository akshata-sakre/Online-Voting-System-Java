# Online Voting System

A console-based Online Voting System developed using Java, JDBC, and MySQL.

## Features
- User Registration
- User Login
- Admin Login
- Add Candidates
- Vote Casting
- Duplicate Vote Prevention
- Result Display

## Technologies Used
- Java
- JDBC
- MySQL

## OOP Concepts
- Classes and Objects
- Constructors
- Encapsulation
- Methods

## Database Tables
- users
- candidates
- admin
